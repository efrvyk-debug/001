CSS

캐스케이딩 스타일 시트(Cascading Style Sheets, CSS)는 웹 페이지의 시각적 스타일과 레이아웃을 정의하는 데 사용되는 스타일 시트 언어이다. CSS는 HTML과 함께 사용되어 웹 페이지의 글꼴, 색상, 간격, 배치 등을 제어하며, 웹 디자인의 일관성과 효율성을 높인다. CSS는 선택자(selector)와 속성(property)을 사용하여 특정 HTML 요소에 스타일을 적용하며, 미디어 쿼리(media query)를 통해 다양한 장치와 화면 크기에 맞게 반응형 디자인을 구현할 수 있다.
