1989년, 팀 버너스리가 CERN에서 제안한 하이퍼텍스트 기반의 정보 관리 시스템
1990년, 월드 와이드 웹(WWW)이라는 이름으로 구현
1991년, 일반에 공개
1993년, 모자익(Mosaic) 웹 브라우저의 등장으로 대중화 시작
1994년, W3C(World Wide Web Consortium) 설립
2004년, 웹 2.0(Web 2.0) 개념 등장
2010년대, HTML5와 CSS3 표준화
현재, 웹은 인터넷의 핵심 서비스로 자리잡음
------------------------------------------------

**구글등은 html 태그를 검색한다**
head: 문서의 메타데이터(문서 제목, 문자 인코딩, 스타일시트, 스크립트 등)
body: 문서의 실제 내용(텍스트, 이미지, 링크, 비디오 등)
html: 문서의 루트 요소
title: 문서의 제목
meta: 문자 인코딩, 키워드, 설명 등 메타데이터 정의
a: 하이퍼링크 생성
